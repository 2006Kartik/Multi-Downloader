Theory:
A linear queue using array is a linear data structure that follows the FIFO (First In First Out) principle, where the first element inserted is the first one to be removed. In this implementation, an array is used to store elements, and two pointers, front and rear, are maintained to keep track of deletion and insertion positions respectively. The enqueue operation inserts an element at the rear end of the queue, while the dequeue operation removes an element from the front end. Initially, both front and rear are set to -1 to indicate an empty queue. Although this method is simple and easy to implement, it suffers from memory wastage, as the empty spaces created after deletion cannot be reused, which leads to the need for a circular queue.
  

Enqueue operation:                                            Dequeue operation:

                           
     
Algorithm for ENQUEUE (Insertion)
1. If REAR == MAX - 1
      Print "Queue Overflow"
      Exit
2. If FRONT == -1
      Set FRONT = 0
3. REAR = REAR + 1
4. QUEUE[REAR] = ITEM
5. Print "Item inserted"

Algorithm for DEQUEUE (Deletion)
1. If FRONT == -1 OR FRONT > REAR
      Print "Queue Underflow"
      Exit
2. ITEM = QUEUE[FRONT]
3. FRONT = FRONT + 1
4. Print "Item deleted"

Algorithm for DISPLAY
1. If FRONT == -1 OR FRONT > REAR
      Print "Queue is Empty"
      Exit
2. For i = FRONT to REAR
      Print QUEUE[i]

 #define MAX 50 
 int queue[MAX]; 
 int front = -1, rear = -1; 
 void enqueue(int value)  
{    
     if (rear == MAX - 1)  
    {        

         printf("\nQueue Overflow!"); 
    } 
    else 
   {        
      if (front == -1)           
      front = 0;         
      rear++;      
      queue[rear] = value;        
      printf("\n%d inserted into queue", value); 
    } 
}  
void dequeue() 
 {   
    if (front == -1 || front > rear)  
   {     
      printf("\nQueue Underflow!"); 
   }
   else 
  {        
    printf("\n%d deleted from queue", queue[front]);         
    front++; 
  } 
} 
void display()  
{ 
    int i;    
    if (front == -1 || front > rear)  
   { 
        printf("\nQueue is Empty!"); 
    }  
    else
    {        
        printf("\nQueue elements:\n");        
        for (i = front; i <= rear; i++)           
        printf("%d ", queue[i]); 
     } 
 }
