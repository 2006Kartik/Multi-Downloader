Aim:  Array Implementation Of Circular Queue.
Objective:
To understand the concept and working of a circular queue using array representation.
To implement enqueue, dequeue, and display operations using a menu-driven switch case  program in C.
To handle overflow and underflow conditions efficiently for optimal memory utilization.
Theory:
   A circular queue using array is a linear data structure that follows the FIFO (First In First Out) principle, where the last position of the array is logically connected to the first position to form a circle. Unlike a simple linear queue, a circular queue efficiently utilizes memory by reusing the vacant spaces created after deletion. The queue operations such as enqueue, dequeue, and display are performed using two pointers, front and rear, and the program uses a menu-driven switch-case structure to allow user interaction. Overflow and underflow conditions are properly handled, making the circular queue suitable for applications like CPU scheduling, buffering, and memory management.                                  
 
        Algorithms:
      1.Enqueue Operation
Read the value to be inserted.
If (rear + 1) % MAX == front, then
                 Print "Queue Overflow"
     Go back to menu.
      3.Else
If front == -1, then
Set front = 0
Set rear = 0
Else
Set rear = (rear + 1) % MAX
     4.  Insert the value at queue[rear].

6.  Return to menu.
      2. Dequeue Operation
If front == -1, then
Print "Queue Underflow"
Go back to menu.
Else
Display the element at queue[front] as deleted.
If front == rear, then
Set front = -1
Set rear = -1
Else
Set front = (front + 1) % MAX
Return to menu.

      3. Display Operation
          1.  If front == -1, then
   Print "Queue is Empty".
         2.  Else
        Set i = front
  Repeat
Print queue[i]
If i == rear, then stop loop
Else set i = (i + 1) % MAX
        3. Return to menu.

CODE:
#include <stdio.h> 
#include <conio.h> 
 
#define MAX 5 
 
int queue[MAX]; 
int front = -1, rear = -1; 
 
void enqueue(int value) 
 {    
     if ((rear + 1) % MAX == front)  
    {         
         printf("\nQueue Overflow!"); 
    }  
    else 
   {        
      if (front == -1)  
     {           
        front = 0;           
       rear = 0; 
    }  
   else  
   {             
        rear = (rear + 1) % MAX; 
    } 
        queue[rear] = value;         
        printf("\n%d inserted", value); 
    } 
} 
 
void dequeue() 
 {    
     if (front == -1)  
    {         
         printf("\nQueue Underflow!"); 
    }  
    else  
     {        
         printf("\n%d deleted", queue[front]);        
         if (front == rear)  
    {            
        front = rear = -1;                                                       // Reset queue         
     }
     else
    {            
        front = (front + 1) % MAX; 
     } 
    } 
}  
void display()
 { 
    int i;     
    if (front == -1)  
   {         
       printf("\nQueue is Empty!"); 
    }  
    else 
     {        
          printf("\nQueue elements:\n");        
           i = front;         
           while (1) 
          {             
              printf("%d ", queue[i]);            
               if (i == rear)                
               break;          
               i = (i + 1) % MAX; 
        } 
    } 
}
